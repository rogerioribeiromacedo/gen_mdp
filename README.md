# GenMDP
 Generator of mdp (Molecular dynamics parameters) file for Gromacs.

## Dependencies
To use the script a few Python 3 dependencies are needed, for example:
- [PyQt5](https://pypi.org/project/PyQt5/)

```
pip install PyQt5
```
